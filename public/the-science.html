The Science

What is DNA Data Storage?


Much data can be stored on memory sticks or the cloud, DNA can also be used to store digital information. DNA data storage is an emerging technology that allows digital information to be encoded into laboratory created DNA.

Computers process and store digital information using binary code, a language consisting of only 0’s and 1’s. Information stored in DNA works in a similar way. DNA is made up of four chemical building blocks, or ‘DNA nucleotides’. These are: adenine (A), guanine (G), cytosine (C) and thymine (T). The sequence of these blocks determine the biological instructions that they carry. For example, ATCGTT might instruct for blue eyes, while ATCGCT might instruct for brown.

DNA data storage involves converting the binary code of a digital file into DNA nucleotide sequences on a computer.

1. Storing the data

Between the Lines uses DNA data storage, an emerging storage medium for digital data.

The original text document was encoded and synthesised as DNA by Twist Bioscience. The sequencing process to retrieve the original data was performed by Exeter Sequencing Service based at Exeter University. To create Between the Lines, stories and experiences from the international student community were collected and compiled into a text document. The text document was then digitised and encoded into DNA nucleotide sequences by Twist Bioscience. Using this sequence, scientists at Twist then constructed synthetic DNA according to the nucleotide sequence.

The resulting synthetic DNA was delivered in small, airtight metal capsules (see image, capsule photographed next to a one penny coin for scale). Each capsule contains close to 200,000 copies of the original text document stored in dried form.
